# First Hexlet Project - Brain Games

## Prerequisites

You need to [install `poetry`](https://python-poetry.org/docs/#installing-with-pipx) for developing this project.

## Setup

1. clone repository
1. `make install`

To install it as Python package, run:

```console
$ make package-install
```
